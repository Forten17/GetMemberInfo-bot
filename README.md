# GetMemberInfo-bot

Discordサーバー内のメンバー情報を取得するPython製botです。

[![MIT License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

## 機能

- ユーザー名からメンバーの参加日やロールを取得
- コマンド例:
  - `!get`

## 動作環境

- Python 3.8 以上
- discord.py
